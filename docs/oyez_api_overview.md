# Oyez API Data Overview

> Generated by Claude - January 2026

Summary of Oyez API structure and case data.
See also: [Audio Formats](oyez_audio_formats.md)

## Overview

- **Total Cases**: 8,477
- **Term Range**: 1850-2025 (75 unique court terms)
- **Total Audio Recordings**: 12,087

## Data Pipeline

```text
scrape index  →  .cache/index/  →  process index  →  cases_index.json
                                                                    ↓
scrape cases     ←  reads case URLs  ←─────────────────────────────┘
     ↓
.cache/cases/api.oyez.org/raw/*.json  (8,477 case detail files)
```

## Case Data Structure

Each case JSON contains the following top-level fields:

| Field | Description |
|-------|-------------|
| `ID` | Unique case identifier |
| `name` | Case name (e.g., "Roe v. Wade") |
| `href` | API URL for the case |
| `docket_number` | Court docket number |
| `term` | Supreme Court term (year) |
| `first_party` / `second_party` | Party names |
| `timeline` | List of case events with dates |
| `facts_of_the_case` | Factual background (HTML) |
| `question` | Legal question presented |
| `conclusion` | Court's conclusion |
| `advocates` | List of attorneys who argued |
| `oral_argument_audio` | **Audio recordings of arguments** |
| `opinion_announcement` | **Audio of opinion announcements** |
| `decisions` | Voting records with justice votes |
| `written_opinion` | Links to written opinions |
| `heard_by` / `decided_by` | Court composition |

| Metric | Value |
|--------|-------|
| Total recordings | 3,517 |
| Cases with audio | 3,457 |

### Dissenting and Concurring Opinions

Justices may read separate opinions from the bench, recorded as
**multiple parts** of the opinion announcement.

**Example: Glossip v. Gross (2015)** - 4 audio parts:

| Part | Speaker | Role |
|------|---------|------|
| 1 | Roberts | Majority announcement |
| 2 | Sotomayor | Dissent |
| 3 | Breyer | Dissent |
| 4 | Scalia | Concurrence |

The `opinion_type` field in votes identifies authors: `majority`,
`dissent`, `concurrence`, `special concurrence`, `plurality`, or `none`.

| Type | Description |
|------|-------------|
| majority opinion | Most common |

| Vote | Count |
|------|-------|

## Unavailable Audio

Some oral arguments are marked unavailable with `public_note`:

- "The audio for this oral argument is missing"
- "This case was dismissed before oral arguments"
- "This case was settled before arguments"
- "This argument audio is not available due to its exclusion from the
  National Archives"
