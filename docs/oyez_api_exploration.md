# Oyez API Data Exploration

> Generated by Claude - January 2026

This document summarizes the findings from exploring the cached Oyez API data.

## Overview

- **Total Cases**: 8,477
- **Term Range**: 1850-2025 (75 unique court terms)
- **Total Audio Recordings**: 12,087

## Data Pipeline

```text
scrape index  →  .cache/index/  →  process index  →  cases_index.json
                                                                    ↓
scrape cases     ←  reads case URLs  ←─────────────────────────────┘
     ↓
.cache/cases/api.oyez.org/raw/*.json  (8,477 case detail files)
```

## Case Data Structure

Each case JSON contains the following top-level fields:

| Field | Description |
|-------|-------------|
| `ID` | Unique case identifier |
| `name` | Case name (e.g., "Roe v. Wade") |
| `href` | API URL for the case |
| `docket_number` | Court docket number |
| `term` | Supreme Court term (year) |
| `first_party` / `second_party` | Party names |
| `timeline` | List of case events with dates |
| `facts_of_the_case` | Factual background (HTML) |
| `question` | Legal question presented |
| `conclusion` | Court's conclusion |
| `advocates` | List of attorneys who argued |
| `oral_argument_audio` | **Audio recordings of arguments** |
| `opinion_announcement` | **Audio of opinion announcements** |
| `decisions` | Voting records with justice votes |
| `written_opinion` | Links to written opinions |
| `heard_by` / `decided_by` | Court composition |
| `citation` | Case citation details |
| `location` | Geographic information |
| `related_cases` | Links to related cases |
| `justia_url` | Link to Justia case page |
| `argument2_url` | Link to Argument 2.0 analysis blog |

## Case Media API Response

When fetching a `case_media` URL, the API returns detailed audio data:

### Audio File Formats

Each recording is available in 3 formats:

| Format | MIME Type | S3 Bucket | Use Case |
|--------|-----------|-----------|----------|
| MP3 | `audio/mpeg` | `oyez.case-media.mp3` | Standard playback |
| OGG | `audio/ogg` | `oyez.case-media.ogg` | Open format |
| HLS | `application/x-mpegURL` | `oyez.case-media.hls` | Streaming |

**Example media_file array:**

```json
{
  "media_file": [
    {
      "id": 50640,
      "mime": "audio/mpeg",
      "size": 13136048,
      "href": "https://s3.amazonaws.com/oyez.case-media.mp3/case_data/1996/95-7452/19961104a_95-7452.delivery.mp3"
    },
    {
      "id": 72426,
      "mime": "audio/ogg",
      "size": 33055710,
      "href": "https://s3.amazonaws.com/oyez.case-media.ogg/case_data/1996/95-7452/19961104a_95-7452.ogg"
    },
    {
      "id": 72427,
      "mime": "application/x-mpegURL",
```

## Unavailable Audio

Some oral arguments are marked as unavailable with `public_note` explanations:

- "The audio for this oral argument is missing"
- "This case was dismissed before oral arguments"
- "This case was settled before arguments"
- "This was a decree. No oral argument took place"
- "This argument audio is not available due to its exclusion from the
  National Archives"

## URL Patterns Summary

| Resource | URL Pattern |
|----------|-------------|
| Case list | `https://api.oyez.org/cases?page={n}&per_page={n}` |
| Case detail | `https://api.oyez.org/cases/{term}/{docket}` |
| Oral argument | `https://api.oyez.org/case_media/oral_argument_audio/{id}` |
| Opinion audio | `https://api.oyez.org/case_media/opinion_announcement_audio/{id}` |
| MP3 files | `https://s3.amazonaws.com/oyez.case-media.mp3/...` |
| OGG files | `https://s3.amazonaws.com/oyez.case-media.ogg/...` |
| HLS streams | `https://s3.amazonaws.com/oyez.case-media.hls/...` |
